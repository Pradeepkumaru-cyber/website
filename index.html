My Work

StealthESP Injector

The StealthESP Injector is a compact hardware device built using the ESP-12S Wi-Fi module and ATmega32U4 microcontroller to perform HID (Human Interface Device) injection attacks for ethical hacking purposes. It allows remote payload delivery and real-time script execution, making it a powerful tool for security testing and demonstrating potential USB-based vulnerabilities.

Reverse Web Shell Generator

This project is a web-based tool designed to generate reverse shell payloads for multiple platforms (Windows, Linux, macOS). It enables users to host payloads, automatically set up listeners, and execute shell connections over HTTP, helping ethical hackers simulate real-world attacks and test network defenses efficiently.

AWS VPC Setup

This project involved designing and deploying a secure and scalable Virtual Private Cloud (VPC) in Amazon Web Services (AWS). The setup included creating custom subnets, route tables, internet gateways, and NAT gateways to enable both public and private access. Security groups and network ACLs were configured to control inbound and outbound traffic, ensuring a robust network architecture for hosting cloud-based applications securely.
